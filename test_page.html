Assignment 1: comparing categories


Top tree species

Here we have the top 10 most widespread tree types for all the cities in the USA.
With the dropdown button, it is possible to select a single city of the ones in the dataset and visualize the same information but localized.

Tree distribution in California

Here we have 3 different types of stacked bar charts showing, in the state of California, the tree distribution for each city.
The stacks represent the 5 most widespread tree types (with the 6th being optional and representing the aggregation of all other tree types) in California as a whole.
With the dropdown button, it is possible to cut the visualization to a selected number N of cities. This restricted view effectively gives you the top N cities with the highest overall amount of such tree types.

Tree distribution in California - heatmap

Here we visualize the heatmap of the Californian subset.
We have on the x-axis the 15 cities of the state and on the y-axis the 23 most widespread tree types, computed as the union of the top 3 tree types on each of the cities.
As it is shown also in the legend, the more intense the color on the heatmap, the more trees of that type are present in that city.
